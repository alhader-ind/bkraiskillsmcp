# Signup UI Specialist: Growth-Centered Onboarding

Signup flows are the engine of user growth. To achieve "100x" better retention, move beyond "registration" and into **Step-Based Progress**, **Value-Demonstration**, and **Zero-Friction Flows**.

## 1. The Onboarding Orchestration Protocol

Engineer growth-ready signups through the **S.I.G.N.U.P.** (Step, Inspect, Gain, Notify, Unit, Pass) framework:

### Phase A: Multi-Step Decomposition (Step)
Avoid "Form Exhaustion":
-   **Bento-Box Forms:** Break long registrations into 3-4 logical steps (Identity, Profile, Preferences).
-   **Progress Visualization:** Use stepper components to show the user how close they are to completion.

### Phase B: Value-First Interstitial (Gain)
Sell the product during the process:
-   **Benefit Callouts:** Use sidebars to remind users *why* they are signing up.
-   **Social Proof:** Display "Join 10k+ users" metrics during wait-states.

### Phase C: Verification Bridge (Notify)
Handle the "Verification Hook":
-   **OTP/Magic Link UX:** Design clear, high-contrast screens for "Check your email" states.
-   **Resend Logic:** Provide smart countdowns before allowing verification resends.

### Phase D: Zero-Friction Handover (Pass)
The "First Mile" experience:
-   **Immediate Redirect:** Send users straight to a personalized "Welcome" or "Setup" page, not just the homepage.
-   **Default Sanity:** Pre-fill fields where possible via referral or invite data.
